setDebugInfo

This task sets the internal Javaâ„¢ compilation debug level, and returns the current settings.

Parameters

Attribute Description Required
DebugInfo Changes all 3 settings No (value unchanged) (may be true or false)
LineNumber Line Number debug information No (value unchanged) (may be true or false)
LocalVariable Local Variable symbol table No (value unchanged) (may be true or false)
SourceFile Source File name No (value unchanged) (may be true or false)
PropertyName Property Name to receive current settings No, default is DebugInfo

Examples

  • Retrieve and display current debug info settings: 
    <setDebugInfo />
<echo message="current settings: ${DebugInfo}" />
  • Set the individual settings to false, then do builds, then set all settings to true: 
    <setDebugInfo
   LineNumber="false"
   LocalVariable="false"
   sourceFile="false" />
<echo message="current settings: ${DebugInfo}" />
... do builds here ...
