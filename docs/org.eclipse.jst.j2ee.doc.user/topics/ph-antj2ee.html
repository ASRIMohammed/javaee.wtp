Ant tasks for J2EE

  • autoAppInstall
    This task uses the WebSphere® Rapid Deploy feature to install an application.
  • AppClientExport
    This task performs the same operation as the Application Client export wizard for exporting an Application Client Project to an Application Client JAR file.
  • EARExport
    This task performs the same operation as the EAR file export wizard for exporting an Enterprise Application Project to an EAR file.
  • UtilJar
    (DEPRECATED) This task compresses source and/or build output of a Java™ project into a JAR file and places the JAR file in an Enterprise Application Project.
  • WARExport
    This task performs the same operation as the WAR file export wizard for exporting a Web Project to a WAR file.
